
# :new_moon_with_face: 내 웃음 돌리도 FrontEnd
>연속으로 나오는 재밌는 짤방과 더불어 다이나믹한 장치들을 통해 웃음을 유발하고 승패를 가리는 웹 기반 게임 서비스입니다.
<br>

## :exclamation: About
>돌리도는 WebRTC 기술을 활용한 화상채팅 기반의 웃음참기 게임입니다. AI가 카메라를 통해 유저의 표정을 인식해 웃을 경우 플레이어의 체력을 깎아 승자를 가립니다. 게임 내에는 플레이어들을 즐겁게 하기 위한 다양한 이벤트가 준비되어 있습니다.
<br>
